# 🧾`I18N-001 日本語/英語ローカライズ`

## 🎯背景/目的

- VS Code の表示言語に合わせてラベル/メッセージを切り替え、利用者の理解負荷を下げる。

## 📌要件

- 5.10

## 🛠️スコープ / 作業内容

- `package.nls.json` / `package.nls.ja.json` でビュー/コマンドのラベルを定義する
- 実行時メッセージは `vscode-nls` を用いて言語切替に対応する
- 既存のメッセージ出力箇所（`src/extension.ts` など）をローカライズ対応へ置換する
- UI/イベント/保存先: 表示言語に応じてラベル/通知が切り替わる（保存先なし）
- 参照: `docs/01_requirements.md`, `docs/02_architect.md`

## ✅AC（受け入れ基準）

- [x] [A11y/I18N] VS Code の表示言語が `ja` の場合は日本語、それ以外は英語で表示される
- [x] [UI/UX] 画面内のラベル/通知で言語が混在しない
- [x] [状態/エラー] ローカライズキーが欠落している場合は英語へフォールバックする
- [x] [テスト] ローカライズキーの解決結果がテストで検証できる
