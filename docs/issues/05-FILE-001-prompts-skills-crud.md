## 🛠️スコープ / 作業内容

- 追加/削除/リネームのコマンドと InputBox フローを実装する
- ルートフォルダ自動作成、拡張子付与、禁止文字置換、重複名回避を実装する
- 削除/上書き時の確認ダイアログを実装し対象名を表示する
