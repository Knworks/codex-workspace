# 🧾`TEST-001 同期機能のテスト追加`

## 🎯背景/目的

- 同期機能の回帰を防止するため。

## 📌要件

- 同期対象の列挙と相互同期処理をユニットで検証する。
- 同期ボタンの表示/非表示と確認ダイアログの分岐を統合で検証する。
- 最終更新日時の比較、隠し除外、エラー時スキップを検証する。
- 削除同期の判定と実行を検証する。
- 削除同期メタ情報の保存/削除を検証する。
- 既存のテスト構成に合わせて追加する。
