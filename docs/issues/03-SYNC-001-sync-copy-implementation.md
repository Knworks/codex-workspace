# 🧾`SYNC-001 同期コピー処理の実装`

## 🎯背景/目的

- 同期ボタン押下時に対象ファイルを設定フォルダへコピーできるようにするため。

## 📌要件

- 確認メッセージを表示し、OK の場合のみコピーを実行する。
- Codex Core は `.codex/AGENTS.md` と `.codex/config.toml` を同期する。
- Prompts / Skills / Templates は各ルート配下のファイルを同期する。
- 上書きコピーとする。

## 🛠️スコープ / 作業内容

- 同期対象の列挙とコピー処理を実装する。
- 確認メッセージをローカライズ対応で追加する。
- 同期完了時のエラー/例外処理を整理する。

## ✅AC（受け入れ基準）

- [ ] [機能] 確認ダイアログで OK を選択した場合のみコピーされる。
- [ ] [機能] Core/Prompts/Skills/Templates それぞれの対象が正しく同期される。
- [ ] [機能] 既存ファイルは上書きされる。
- [ ] [UI/UX] 確認メッセージが日本語/英語で表示される。
- [ ] [テスト] 同期対象の列挙とコピー処理のテストが追加されている。

## 🔗依存関係

- DependsOn: CONFIG-001, UI-001

## 🧪テスト観点

- (ユニット / リクエスト)
- 検証方法：確認ダイアログの分岐とコピー結果を確認する。
