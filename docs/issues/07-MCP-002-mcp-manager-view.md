# 🧾 MCP-002 MCP Manager View

## 🎯 背景/目的

- MCP サーバーの詳細確認、追加、編集、削除を Explorer のトグル操作から分離して管理画面で扱う。

## 📌 要件

- MCP Manager View は単一 WebviewPanel とする。
- 左側ペインに一覧、右側ペインに編集フォームを表示する。
- ON/OFF トグルのみ即時保存し、フォーム編集は保存ボタンで反映する。

## 🛠️ スコープ / 作業内容

- MCP Manager View と Webview メッセージ処理を追加する。
- 検索、選択、追加、削除、保存、キャンセルを実装する。
- 未保存状態での操作確認を実装する。
- ON/OFF トグル後に MCP Explorer と Manager View を更新する。
- 関連テストを追加する。

## ✅ AC（受け入れ基準）

- [ ] [機能] MCPサーバー名で一覧を絞り込みできる。
- [ ] [UI/UX] 左ペインに一覧と追加削除ボタン、右ペインに編集フォームを表示できる。
- [ ] [状態/エラー] 未保存状態で別操作を行う場合に保存、破棄、キャンセルを選択できる。
- [ ] [機能] 追加は保存ボタンまで `config.toml` に反映しない。
- [ ] [テスト] Webview メッセージ処理、未保存確認、トグル即時保存をテストする。
