# 🧾`FILE-002 Templates Explorerとテンプレ適用`

## 🎯背景/目的

- テンプレートを選択して雛形を適用できるようにし、作成作業を効率化する。

## 📌要件

- 5.1
- 5.6

## 🛠️スコープ / 作業内容

- Templates Explorer を追加し固定ルート `codex-templates` 配下の階層を表示する
- テンプレート選択 UI を実装し、新規ファイル作成時に内容を反映する
- 隠しファイル（`.` 始まり）はテンプレート候補から除外する
- UI/イベント/保存先: 作成操作でテンプレートを選択（保存先: `~/.codex/codex-templates`）
- 参照: `docs/01_requirements.md`, `docs/02_architect.md`

## ✅AC（受け入れ基準）

- [x] [機能] テンプレートファイルが存在する場合のみ選択 UI が表示される
- [x] [機能] テンプレート選択後に内容が新規ファイルへ反映される
- [x] [状態/エラー] 隠しファイル（`.` 始まり）はテンプレート候補から除外される
- [x] [UI/UX] Template Explorer からテンプレートファイルを開いて編集できる
- [x] [テスト] テンプレート適用と候補フィルタがテストで検証できる
